This is server for MySkills API App - please visit APP or check the informations below.

DESCRIPTION

This API is used as my portfolio project. You can fetch data from MongoDB database to see what my skills are and get my contact information

GET ALL SKILLS

Use route: "/api/skills". Returns array of JSON objects.

GET SKILL BY ID

Use route "/api/skills/:id". Returns JSON object.

GET SKILLS WITH PARAMS

Use route "/api/skills/?name=name_of_skill&type=type_of_skill&tag=skill_with_tag". Returns array of JSON object/s. Check usuall names/types/tags
